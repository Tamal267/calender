Open your linux terminal || termux , then - 

~ git clone https://github.com/Tamal267/calender

~ cd calender

~ chmod +x c.sh

~ ./c.sh

You will see the current gregrian month

If you want to see other calender like calender to November 2018 , then

~ ./cmanual.sh

~ Enter the number of year & month (ex: 2018 11)

You will see the output
